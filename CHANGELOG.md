# Changelog

## 0.1.0-rc1

### axt-peek

- Add the `axt-peek` MVP with human, JSON, JSONL, plain, and ACF agent output modes.
- Add `axt.peek.v1` JSON schema generation and validation coverage.

### Release

- Prepare the cargo-dist release pipeline for the v0.1.0 release candidate.
- Add publishable package metadata for the crates required by `cargo install axt-peek`.
