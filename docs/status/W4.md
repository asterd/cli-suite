# Wave 4 Status Report

## What was implemented
- Reused the existing `axt-peek` visible metadata collection for ignored-path comparison and normalized entry paths before downstream rendering.
- Added shared text-byte classification in `axt-fs` and adopted it in `axt-slice` and `axt-ctxpack`.
- Added Low-severity polish for `axt-run` cleanup mtime handling, `axt-outline` safe source slicing, `axt-drift` symlink skip diagnostics, and `axt-logdx` timestamp parsing.
- Added regression coverage for stable `axt-bundle --max-files` truncation and `axt-logdx` fractional epoch plus mismatched RFC3339 delimiter parsing.
- Added nightly/manual OS smoke CI jobs and shared smoke scripts for Unix and Windows.

## What was not implemented and why
- Production regression fixtures were not added because no new production bug report was provided in this session.
- A fully exact single physical filesystem pass for `axt-peek` ignored counting was not implemented because the current ignore walker does not expose skipped ignored entries; this session removed the duplicate visible metadata walk and keeps the raw comparison pass.

## Deviations from the spec
- None. The changes stay within the existing command surface and schemas.

## How to run tests
```bash
cargo fmt --all --check
cargo clippy --workspace --all-targets -- -D warnings
cargo test --workspace
```

## How to run the release pipeline locally
```bash
bash tests/smoke/unix.sh
pwsh ./tests/smoke/windows.ps1
```
